# 0104: Ei Context Exporter

**Status**: PENDING
**Depends on**: 0101

## Summary

Export Ei's understanding of the human (facts, traits, topics, people, current state) to a markdown file that can be injected into CLAUDE.md or project AGENTS.md files.

## Acceptance Criteria

- [ ] Exports to `~/.ei/context-export.md`
- [ ] Includes current emotional state / sentiment trends
- [ ] Includes active topics (exposure_current > threshold)
- [ ] Includes recent facts (last_updated within N days)
- [ ] Includes people in context
- [ ] Includes communication preferences (from traits)
- [ ] Format: Markdown with clear section headers
- [ ] Auto-updates on state change (debounced)
- [ ] CLI command: `ei export-context [--output PATH]`
